AI Medical Assistant is a Retrieval-Augmented Generation (RAG) application that provides accurate, context-aware medical answers by combining semantic search with large language models.

Instead of relying solely on LLM knowledge, the system retrieves relevant medical information from the Endee vector database and uses it to generate grounded, reliable responses.

🔹 Key Highlights:
- Uses RAG to reduce hallucinations
- Semantic search with sentence-transformers
- Fast vector retrieval using Endee
- Context-based answer generation via Claude LLM
- Transparent responses with source chunks

This project demonstrates a real-world AI architecture used in production systems for healthcare, search, and knowledge assistants.
